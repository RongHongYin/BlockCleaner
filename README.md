# BlockCleaner（区块清除器）

一个用于批量清理方块的 Fabric MOD。  
核心思路是：在指定范围和高度区间内，按规则逐格扫描并破坏方块，将掉落物自动输出到容器。

## 版本与依赖

- Minecraft: `1.21.11`
- Fabric Loader: `>=0.18.6`
- Java: `>=21`
- Fabric API: 必需

## 主要功能

- 提供一个可交互方块：`区块清除器`
- 两种清理模式：
  - **生存模式**：按方块采掘逻辑处理，使用工具并产出掉落物
  - **创造模式**：直接清除方块，不产出掉落物
- 两种范围计算模式：
  - **中心模式**：以机器所在区块为中心扩展（范围仅允许奇数）
  - **左上角模式**：以机器所在区块作为左上角起点向右下扩展（范围可为任意 1-99）
- 方向可选：
  - 向上清理（从机器上方开始）
  - 向下清理（从机器下方开始）
- 可设置目标 Y，用于限制清理高度终点
- 速度模式：
  - **固定**：按每秒方块数执行
  - **原版**：按方块硬度和工具挖掘速度计算间隔
- 掉落物黑名单（全局），可将指定物品过滤不输出

## 合成配方

工作台 3x3：

- 第一行：`钻石块` `下界之星` `钻石块`
- 第二行：`下界之星` `下界合金块` `下界之星`
- 第三行：`钻石块` `下界之星` `钻石块`

## 使用方法

1. 放置区块清除器并右键打开界面。
2. 准备容器：
   - 面向机器正面时，**左侧为输入**、**右侧为输出**。
   - 会自动搜集相连的容器网络（不止一格）。
3. 在界面配置参数（模式、方向、范围模式、范围、目标 Y、速度等）。
4. 点击“启动”开始清理。

## 输入 / 输出规则

- **输入侧（左）**：
  - 生存模式下需要工具（按方块类型匹配镐/斧/锹）。
  - 遇到水/岩浆时，会尝试消耗可放置方块进行填充。
- **输出侧（右）**：
  - 存放掉落物；若放不下，多余掉落会在世界中掉落。

## 参数说明

- **模式**：创造 / 生存
- **保留 1 耐久**：开启后，工具剩 1 耐久时不再继续消耗
- **方向**：向上 / 向下
- **目标 Y**：清理终止高度
- **范围模式**：
  - 中心：范围为奇数（1-99）
  - 左上角：范围为 1-99（无奇偶限制）
- **范围**：以“区块数（chunk）”计
- **速度模式**：固定 / 原版
- **速度**：固定模式下生效（10 的倍数）

## 安全与过滤

- 不会清理以下方块/目标：
  - 基岩、屏障、命令方块（含连锁/循环）、结构方块、拼图方块
  - 任意带方块实体（BlockEntity）的方块
  - 机器本体所在位置
- 黑名单中的掉落物不会被输出（直接过滤）

## 黑名单机制说明

- 黑名单是**全局共享**（同一存档内所有区块清除器共用）
- 数据保存文件：世界根目录下 `blockcleaner-global-blacklist.txt`
- 支持在 GUI 中添加、移除和清空黑名单

## 注意事项

- 生存模式依赖输入侧提供可用工具，否则会停在无法处理的方块上。
- 固定速度过高时，可能增加服务器负载，请按机器性能调节。
- 清理结束后机器会自动停止，并广播状态消息。
